# CULTIA: Investor-Grade Cultural Heritage Platform

## 🌟 Overview
CULTIA is a cutting-edge educational and cultural platform dedicated to preserving and celebrating global heritage through engaging, interactive experiences. Our mission is to make cultural learning accessible, fun, and impactful for users of all ages.

---

## ✨ Key Features

### 1. **Story Creator & Moderation System** (NEW!)
- **User-Generated Content (UGC)**: Empowers community members to share their own cultural stories, traditions, and knowledge
- **Full Moderation Workflow**: Admin panel for reviewing, approving, rejecting, and publishing stories
- **Status Tracking**: Pending → Approved → Published pipeline
- **Quality Control**: Ensures only appropriate, high-quality content reaches users

### 2. **Subscription Tiers** (Monetization Ready!)
- **Free Tier**: Basic access to core features
- **Premium Tier**: Unlocks exclusive content, advanced features, ad-free experience
- **Family Tier**: Multi-user access for families
- **Educator Tier**: Bulk licenses for schools and institutions

### 3. **Cultural Ambassadors Program**
- Users can apply to become verified cultural ambassadors
- Moderated application process
- Ambassadors can contribute premium content

### 4. **Marketplace for Artisans & Workshops**
- Platform for cultural artisans to sell products and offer workshops
- Commission-based revenue model (monetization opportunity)

### 5. **Cultural Impact Score**
- Tracks user engagement and contributions to cultural preservation
- Encourages long-term platform usage

### 6. **Adaptive Learning**
- Personalized content recommendations based on user preferences and learning style
- Keeps users engaged and coming back for more

### 7. **Robust Gamification System**
- Points, badges, achievements, and leaderboards
- Widget system for interactive learning games
- Proven to increase user retention

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- PostgreSQL 13+
- Node.js (for frontend tools, optional)

### Installation & Setup
1. Clone the repository
2. Create a PostgreSQL database named `cultia`
3. Set the `DATABASE_URL` environment variable
4. Run `python backend/init_db.py` to initialize the database
5. Start the backend with `python backend/api.py`
6. Access the platform at `http://localhost:5000`

### Admin Access
- Create a user account, then set the role to 'admin' in the database
- Access the admin panel at `http://localhost:5000/bot/admin.html`
